# Бриф для соавторов

## Зачем это делается

Нужно собрать не абстрактный манифест про ИИ, а рабочую программу для Montelibero:
- с проблематизацией;
- с понятными механизмами;
- с пилотом;
- с критериями оценки;
- с местом ИИ-агентов в экосистеме.

## Что особенно интересно получить от соавторов

1. Какие реальные проблемы Montelibero может решать ИИ-слой уже сейчас?
2. Где граница между полезной автоматизацией и опасной симуляцией деятельности?
3. Какие роли агентов и людей нужно развести жёстко?
4. Какие метрики покажут, что программа реально работает?
5. Как организовать память, координацию и обратную связь так, чтобы система не рассыпалась?

## Формат вклада

Лучше всего присылать вклад в одном из форматов:
- готовый кусок текста;
- список тезисов по разделу;
- критику текущего драфта с предложением замены;
- issue с конкретным вопросом/риском/гипотезой.
